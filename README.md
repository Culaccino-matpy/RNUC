# Residual-learning based Non-uniformity Correction Net（RNUCnet）
## 2022.4

本代码是本人毕业设计
毕设题目：基于卷积神经网络的红外图像非均匀性校正的研究。

根据两个残差块进行级联的方式提出一种基于残差学习的红外图像非均匀性校正的方法：Residual-learning based Non-uniformity Correction Net，简称（RNUC）
